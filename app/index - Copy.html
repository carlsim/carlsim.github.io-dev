ABOUT ME_

Name: Carl Sim

Occupation: Web Developer, Photographer, UI/UX Designer

Description: A developer, ui/ux designer, and photographer focusing on projects that are beautiful, rich experiences on the web.

SPECIALTY_

photography_

I shoot mostly landscapes, and portraiture but I also have done several event shoots at the Kampai Sake Festival, Ryerson's Alumni Weekend, and the DOTA 2 International viewing at TIFF.

web development_

As a seasoned front-end developer, I create experiences that pushes my skills to create a product that anyone would be proud to use.

I'm proficient in web technologies such as HTML/CSS, CSS-Preprocessors (SASS/LESS), Javascript task runners (Gulp.js), Wordpress, ES6, ES5, VueJS, Vuex, Webpack, CSS methodology like BEM/SMACSS, etc.

ui/ux design_

I love to make things look pretty while at the same time solving the everlasting problem of design. With the use of well thought out layouts, graphical elements, meaningful motion, and user stories of how a user would interact with the product, I can get reach closer to that perfect user experience.
